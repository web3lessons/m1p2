# Основа простейшего блокчейна

## Контент

В этом уроке мы реализуем простейший блокчейн, включая:

- цепочку блоков, связанных хэшами;
- транзакции;
- проверку блоков на соответствие правилам;
- майнинг и награду за добычу блоков.

Реализуем в будущем: электронные подписи.
Не реализуем: работу с сетью и всё, что с ней связано.

## Технологии

Работаем в VS Code.

Для простоты не используем TypeScript, сборку, линтинг, автотесты и прочее.
Эти инструменты рекомендуем к самостоятельному изучению и применению.

Впрочем, некоторая типизация нам здесь, всё же, пригодится: используем JSDoc.
